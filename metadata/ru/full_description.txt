Блокирует нежелательные звонки, SMS и уведомления приложений, не заменяя приложение для звонков/SMS по умолчанию.

Проверка уведомлений (Notification Screening) применяет ваши правила блокировки к RCS, Signal, WhatsApp, электронной почте и любому другому приложению для сообщений, показывающему уведомления, а не только к вашему стандартному приложению SMS, поэтому спам отфильтровывается независимо от того, через какое приложение он приходит. Оповещения (Alerts) дают лёгкий звук/вибрацию/фонарик/пробуждение экрана для разрешённых проверенных уведомлений, вместо повторного полноценного уведомления.

Фильтры:

1. Правила по регулярному выражению для номера и содержания SMS.
2. Контакты
3. Аттестация STIR
4. Повторный звонок
5. Набранный номер
6. Недавно использованные приложения
7. Расписание
8. Проверка уведомлений (заголовок/текст уведомлений выбранных приложений)
